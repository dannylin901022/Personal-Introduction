網頁程式設計

網頁程式設計　CodePen 練習

　　HTML


語法筆記

語法 用途
DOCTYPE 告訴瀏覽器頁面是用什麼標籤語言編成，使瀏覽器正確解讀網頁
style 設計內容樣式，詳細：CSS
< html> 標示文件的開始及結束，所有東西都要包含在裡面
< head> 表示標頭位置，該區不會出現在頁面中
< title> 網頁(分頁)標題
< body> 網頁內容呈現區塊
< h1> 定義字體大小，h1最大；h6小
< p> 放置文字段落，並可在段落中呈現文字，一個< p>為一個段落
< br> 強制換行
< hr> 生成水平線
< a> 超連結，href：指定連結網址；target：指定開啟方式
target=_blank：開在新分頁；target=_parent：當成網頁上的一個畫面；target=_self：以目前頁面開啟
< table> 生成表格
　tr：生成橫列；
　th：標示表格title；
　td：標示表格儲存格
< ul> 製作清單列表
　ul：無序列表；
　ol：有序列表；
　li：標示個別項目

　　CSS


屬性筆記

屬性 用途
background 設定項目背景
　color：設定顏色；
　image：設置背景圖片...
font 設定文字樣式
　size：設定文字大小
　family：設定字體
　去除font，純用color：設定文字顏色
　weight：設定文字粗細
text 設定排版
　decoration：underline 加底線、none 無底線
　align：文字對齊(center置中、left靠左、right靠右)
border 設定邊框
　width：邊框寬度
　style：邊框樣式(單線：solid、雙線：double、虛線：dashed)
　color：邊框顏色
padding 設定內部邊距
margin 設定外部邊距
display 元件顯示
　block：區塊顯示(預設)
　none：不顯示(移除區塊)
　flex：設定flex排版
flex justify-content：橫列對齊方式
align-items：垂直對齊方式
flex-direction：flex排列方式(row：橫向、column：垂直)
CSS動作 hover：滑鼠移動到元件上時動作
active：滑鼠點擊元件時動作

　　Fetch Data


基本語法

var data = {};
await fetch(url,{
　method: "GET",
　headers{
　　"Content-Type": "application/json",
　　Accept: "application/json",
　}
})
.then((response) => {
　return response.json();　　//If data is JSON file
})
.then((response) => {
　data = response;
})
.catch((error) => { })


語法筆記

語法 用途
GET(預設) 用於獲取API所傳輸之資料
POST 透過API傳遞資料給後端，新增資料時可使用，並獲取回傳值
需在headers下方增加body，如body: JSON.stringify(data)，data為要傳輸之資料，須符合傳輸資料之格式
PUT 透過API，修改後端已存在的資料，不會有回傳值
DELETE 透過API，刪除後端資料，不會有回傳值
